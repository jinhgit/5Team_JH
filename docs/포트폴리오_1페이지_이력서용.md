# 명랑회 (Myeongrang-hoe) — 1페이지 이력서용 요약

**캠퍼스 동행 펀딩 웹앱** · 학교 이메일 인증 기반 소모임 모집·참여·채팅·후기 플랫폼  
**역할**: Full-stack (Spring Boot API + React FE 연동·UX 고도화)  
**스택**: Java 17 · Spring Boot 4 · JPA · H2 · React 19 · TypeScript · Vite · Tailwind · Kakao Maps  
**링크**: [PR #3](https://github.com/mju-myongjithon/5Team/pull/3) · [포크](https://github.com/jinhgit/5Team_JH) · [상세 명세](./포트폴리오_명랑회_기능명세.md)

---

## 한 줄 성과
로컬 스토어 데모 수준 앱을 **서버 영속 REST API + FE 하이브리드 연동**으로 전환하고,  
지도 기반 발견·성사–일정–채팅–후기 루프·안전/알림 UX까지 제품 완성도를 끌어올림.

---

## 핵심 기여 (Bullet)

- **인증·회원**: 학교 메일(`*.ac.kr`) OTP, BCrypt 가입/로그인, Bearer 세션, 프로필·위치 API  
- **펀딩 도메인**: CRUD, 참여/취소, 인원 확정, 조기 마감·삭제, 일정 확정, Haversine 주변 조회, 규칙 기반 노쇼 리스크  
- **커뮤니티**: 댓글(중복·IME 버그 수정), 채팅 폴링·미읽음, 후기→햇살지수, 찜·성사임박 알림  
- **FE 아키텍처**: `api.ts` + 서버 우선/로컬 폴백 스토어, AuthBootstrap, 전역 토스트·로딩  
- **미디어**: Multipart 이미지 업로드(`/uploads`), 아바타·커버, 위치 지도 커버 폴백  
- **발견 UX**: 카카오맵 안정화, 클러스터·재생 투어, 거리/날짜/요금/카테고리(쇼핑) 필터, 관심사 가로 스크롤  
- **제품 기능**: 카톡/Web 공유, 온보딩, 신고·차단, 브라우저 알림, Google 캘린더·`.ics`  
- **협업**: 한국어 체크포인트 커밋, PR 본문·기능 로드맵·포트폴리오 문서화
